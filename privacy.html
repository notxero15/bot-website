🔒 Privacy Policy

Last updated: April 2026

1. Information We Collect

Chromify may collect limited data necessary for functionality, including:

  • Server IDs (to manage settings per server)
  • User IDs (for role assignment and permissions)
  • Message IDs (for reaction role tracking)

2. How We Use Data

The collected data is used only for bot functionality, such as managing color roles, storing server configurations, and ensuring features work correctly.

3. Data Storage

Chromify stores minimal data locally or on secure storage systems. We do not sell, share, or distribute user data to third parties.

4. Data Removal

Server administrators may remove bot data at any time by removing the bot from their server. This will automatically delete associated server data.

5. Third-Party Services

Chromify operates using Discord’s API. We are not responsible for Discord’s data handling policies.

6. Security

We take reasonable steps to protect stored data, but no system is 100% secure.

7. Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of Chromify means acceptance of any changes.

8. Contact

For privacy concerns, contact the bot owner via Discord.
