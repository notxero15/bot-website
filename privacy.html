Privacy Policy

Last updated: April 2026

This Privacy Policy explains how Chromify (“the Bot”) collects and uses data when used in Discord servers.

1. Data We Collect

The Bot may collect and store the following information:

  • Discord Server IDs
  • Discord User IDs
  • Message IDs (for reaction role systems)

2. How We Use Data

This data is used only for bot functionality such as:

  • Managing color roles
  • Saving server settings
  • Handling reaction role systems

3. Data Sharing

We do not sell, trade, or share any user data with third parties.

4. Data Storage

Data is stored locally or on secure hosting systems only for bot operation.

5. User Rights

Users may request data removal by removing the bot from their server.

6. Contact

For questions or concerns, join the support server linked on our website.
